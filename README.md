# jd_speed_sign_Part1~jd_speed_sign_Part3
简单粗暴的极速版的分任务版，将总ck数除以3后平均分配成三个任务同时执行.

如果使用请务必禁用其他库的jd_speed_sign脚本.感谢jd_speed_sign原作者。
	
例子 : 有24个ck，则Part1 执行1~8,Part2 执行9~16，Part3 执行17以后剩下的所有ck.
